1. Создание объявления <br>
Отправить запрос `/api/1/item` с корректным телом объявления <br>
{ <br>
 "sellerID": 123123, <br>
 "name": "Товар", <br>
 "price": 1000, <br>
 "statistics": { <br>
   "likes": 0, <br>
   "viewCount": 0, <br>
   "contacts": 0 <br>
 } <br>
} <br>
Ожидаемый результат: <br>
Код: 200 <br>
Тело содержит поля: id , sellerID , name, price, statistics, createdAt <br>
2. Получение объявления по идентификатору <br>
Отправить GET-запрос `/api/1/item/{id}`, подставив id объявления. <br>
Ожидается: <br>
Код ответа 200 <br>
Тело - массив из одного элемента <br>
Элемент содержит все ожидаемые поля <br> 
3. Получение всех объявлений по sellerID <br>
Отправить GET-запрос `/api/1/123123/item` <br>
Ожидается: <br>
Код 200 <br>
В теле ответа непустой массив объявлений <br>
4. Получение статистики по item ID <br>
Отправить GET-запрос `/api/1/statistic/{id}` подставив id полученный при создании объявления <br>
Ожидается: <br>
Код 200 <br>
Массив со статистикой содержащий поля likes, viewCount, contacts <br>
